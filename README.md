# PeptidePro — Pharmaceutical-Grade Peptide Store

Interactive mockup of a peptide e-commerce site targeting US physicians.

## Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Run development server
npm run dev
```

The site will open automatically at `http://localhost:5173`

## Requirements

- **Node.js** 18+ ([download](https://nodejs.org))

## Features

- **Home page** — hero section, trust badges, featured products
- **Catalog** — filterable product grid with search and category filters
- **Product detail** — full specs (CAS, sequence, MW, sizes, pricing)
- **Physician registration** — NPI number, specialty, verification flow
- **AI Chatbot** — click 💬 button, ask about BPC-157, storage, quality testing
- **COA Library** — certificate of analysis access per product

## Build for Production

```bash
npm run build    # outputs to /dist
npm run preview  # preview the production build
```

## Tech Stack

- React 18
- Vite 6
- Google Fonts (DM Sans + Instrument Serif)
